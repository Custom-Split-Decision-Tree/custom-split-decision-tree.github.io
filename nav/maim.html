CSDT

CSDT (Custom Split Decision Tree) is a new decision tree model that allows users to build a decision tree using their custom prediction and split methods. Unlike traditional decision trees, CSDT enables users to customize prediction and split functions based on their specific needs. With these custom inputs, CSDT makes machine learning models more flexible and tailored to individual requirements.

CSDT can be used in a variety of applications. It is especially effective in classification, regression, and optimization-based output-constrained Linear Programming (LP) problems. Its performance is comparable to Scikit-learn's standard decision tree implementation. For instance, when the prediction and split methods of DecisionTreeRegressor or DecisionTreeClassifier are integrated into CSDT with the same parameters, both models yield identical results.

Below is an example implementation of how to create and use a CSDT model: 


from csdt import CSDT

# Step 1: Import libraries and prepare data
import pandas as pd
from sklearn.datasets import make_regression

X, y = make_regression(n_samples=100, n_features=5, noise=0.1, random_state=42)
X_df = pd.DataFrame(X, columns=[f'Feature_{i}' for i in range(X.shape[1])])
y_df = pd.DataFrame(y, columns=['Target'])

# Step 2: Define custom split criteria
def custom_split_criteria(y, X):
    return y.mean(), ((y - y.mean())**2).sum()

# Step 3: Initialize the CSDT model
tree = CSDT(
    max_depth=5,
    min_samples_leaf=10,
    min_samples_split=20,
    split_criteria=custom_split_criteria,
    random_state=42
)

# Step 4: Train the model
tree.fit(X_df, y_df)

# Step 5: Make predictions
predictions = tree.predict(X_df)

print("Predictions:")
print(predictions)
