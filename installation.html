Custom Split Decision Tree Installation

Follow the instructions below to install the Custom Split Decision Tree (CSDT) library. Whether you're using pip or Conda, installation is straightforward and well-supported.

Option 1: Install via pip

You can install the library directly from PyPI using pip. Run the following command in your terminal: 


pip install csdt

Option 2: Clone and Install Locally

Alternatively, you can clone the repository from GitHub and install the library locally. Use the following commands: 


git clone https://github.com/cagla0117/Custom-Split-Decision-Tree.git
cd Custom-Split-Decision-Tree
pip install -e .

Requirements

The CSDT library requires the following dependencies. These will be automatically installed if you use pip:

  • matplotlib==3.8.1
  • numpy==1.26.0
  • pandas==2.1.3
  • scikit-learn==1.3.2
  • graphviz

Conda Environment Setup

If you prefer using Conda, you can set up an environment with all required dependencies using the provided `csdt.yml` file. Follow these steps:

Step 1: Download the `csdt.yml` file

Make sure the `csdt.yml` file is available in your working directory.

Step 2: Create the Environment

Run the following command to create the environment:


conda env create -f csdt.yml

Step 3: Activate the Environment

Activate the newly created environment:


conda activate csdt
